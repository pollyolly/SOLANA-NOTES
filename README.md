### SOLANA-NOTES

### Installation
```
sh -c "$(curl -sSfL https://release.solana.com/v1.17.16/install)"
```
close terminal and run the path
```
export PATH="/Users/roco/.local/share/solana/install/active_release/bin:$PATH"
```

```
$solana-keygen new
```
